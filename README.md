Voting Program (Solana + Anchor)

A simple decentralized voting program built on Solana using the Anchor framework.
The program allows users to:
    - Register as a candidate
    - Register as a voter
    - Cast a single vote for a candidate

Each voter can vote only once, and all data is stored on-chain using PDA accounts.
